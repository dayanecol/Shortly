# Shortly

This is a back-end project about link reduction. 

## Features

- **POST** `/signup`
- **POST** `/signin`
- **POST** `/urls/shorten`
- **GET** `/urls/:id`
- **GET** `/urls/open/:shortUrl`
- **GET** `/users/me`
- **GET** `/ranking`
- **DELETE** `/urls/:id`
